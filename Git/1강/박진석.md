  ## Git이란?
#### 1. 형상 관리 도구  
```
    1.1 버전관리 시스템(VCS- Version Control System) 이라고도 하며 대표적으로 CVS, SVN, GIT 등이 있다
    1.2 SVN과 GIT 의 차이점
        - SVN                            |     - GIT
          중앙서버 업로드                 |       로컬저장소 저장후 서버 업로드
          동시 업로드시 충돌가능성↑        |       Branch와 Merge를 통해 충돌가능성 ↓
          모든작업에 네트워크 사용         |       작업은 로컬, 업로드만 네트워크 사용
          히스토리 관리가 일시적           |       히스토리 관리 용이
```
